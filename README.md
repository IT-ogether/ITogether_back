## ITogether
컴퓨터공학을 전공하거나 IT를 학습하는 사용자들이 어려움 없이 정보를 찾을 수 있도록 도와주고 다양한 IT 정보를 제공하는 프로젝트입니다.

## 개발 기간 및 버전
**2022/03/07 - 2022/06/01**

**V1 (~4/13)**
* 메인 정보
* 디테일 정보
* 카카오 로그인 및 인증
* 프로필

**V2 (~6/1)**
* 후기 크롤링
* 좋아요 기능
* 공유 수 카운트
* 직군 선택
* 관심 직군 기반 추천
* 세부 직군 별 필터링

## 사용 기술 및 환경
Java 11, Spring boot, Gradle, JdbcTemplate, MySQL, AWS
